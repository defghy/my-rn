## 启动

#### Android
由于使用了AndroidX，有些第三方库不支持，需要借助`jetify`修改依赖

###### 前置操作

1. npx jetify
2. Android Studio(>3.2.0) 打开 android
3. gradle sync

###### 日常操作

1. npm run start
2. Android Studio(>3.2.0) 打开 android
3. run
